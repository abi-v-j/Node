# Node


## Installation

1. Clone the repository to your local machine:

    ```bash
    git clone https://github.com/abi-v-j/Node.git
    ```

2. Navigate to the project directory:

    ```bash
    cd Node
    ```
    
3. Installing Node Modules 
    
    ```
    npm i
    ```
4. run backend 
    
    ```
    npm run dev
    ```
